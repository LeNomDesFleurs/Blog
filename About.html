Hi !
I'm currently a student in an audiovisual school in Paris.
I'm learning code by myself, and this website is a way of dipping a toe in html and helps me practice my english.
(It also keeps my mom up to date on what I'm working on)
I'm mostly interested in uncovering new sonic scape by magnifying error in digital signal processes.
For me, this work is only completed by an extensive work on interfaces, keeping the most gnarly delay line network usable by someone with no knowledge of the underlying working process.

Turning a knob should always be fun.

My tools for creating music are Ableton and it's native plugs, superCollider, Modular Synth.
My coding practive started with C++ and VCV Rack, and continued with the JUCE plugin framework, with the goal to make the tools I create available to more people.

I'm starting to get interested by visual, using TouchDesigner for Audiovisual Exposition and Hydra for personal fun.
